# dotVim

vim 8 package

## install

> git clone git@github.com:BruceMaa/dotVim.git ~/.vim

## vim-scripts

1. vim-go

   > git remote add vim-go git@github.com:fatih/vim-go.git
   > 
   > git subtree add --prefix=pack/plugins/start/vim-go vim-go master --squash
